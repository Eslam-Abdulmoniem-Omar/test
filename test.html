Frequently Asked Questions

How is it different from CTRL + S or File -> Save?

UI Replicator gives better results, it is faster, the sources are hotlinked and saving the HTML directly almost never works.

If you get the chance, you can try it on PhotoAI.com and compare it with the showcase video in the "How It Works" section.

Is this tool legal?

Yes! The tool is 100% legal, in the same way screen recording is 100% legal. The fact that it can be used for bad stuff, is a whole different story.

However, it is 100% the responsibility of the user how he/she uses the tool, either for good or for bad. I encourage everyone to use it as a learning and inspiration tool, and not steal somebody else's work.

Is UI Replicator using AI?

Nope! UI Replicator has 0% AI in it, and it works completely on parsing algorithms.

Can it clone Webflow websites?

Yes! We tested UI Replicator on multiple Webflow websites and it worked perfectly.

However, you need to keep in mind that the JS scripts, and possible more things, are served through CDNs and if Webflow decides to block your site, they can do it anytime.

Can it clone Framer websites?

Yes! We tested UI Replicator on multiple Framer websites and it worked perfectly.

However, you need to keep in mind that the JS scripts, and possible more things, are served through CDNs and if Framer decides to block your site, they can do it anytime.

What are the websites that cannot be cloned?

Some pages require authentification, an example is your Twitter / X message page, and since we cannot ask your for yout authentification cookies or tokens for safety reasons, we cannot clone it.

Another type of websites that cannot be cloned are the ones 100% dynamically loaded from CDNs. Best example here is Bubble.io - whenever you open a website made in Bubble, they will load everything from their database, and if the origin of the request doesn't match, the website won't load and you will get an error message.

How many websites can I clone?

You can clone an unlimited number of websites, and you can use it as frequently as you want.
